## 🗓 Study Plan Overview

The plan runs **two parallel tracks** over 6 months, alternating subjects daily so you never study the same thing two days in a row.

```
MON  →  CSCP
TUE  →  AWS
WED  →  CSCP (practice questions)
THU  →  AWS
FRI  →  Review only (Anki / flashcards)
```

## ⚡ The 6 ADHD Study Rules

These rules are the foundation of the plan. Breaking them is why most people quit.

1. **25-min Pomodoro only** — never study longer without a break
2. **One topic per session** — CSCP or AWS, never both in one block
3. **Max 2 sessions per weekday** — consistency beats marathon sessions
4. **3-thing recall after every session** — close everything and write 3 things from memory
5. **Friday = review only** — no new material, Anki flashcards only
6. **Sunday = 10-min weekly plan** — replaces anxiety with clarity

## 📦 CSCP Resource Stack

| Resource | Cost | Why |

| Udemy CSCP Prep Course | ~$15 (on sale) | Video-first, watch at 1.5× speed |
| ASCM Learning Portal | ~$225/yr membership | Official, short modules with built-in quizzes |
